# Privacy Policy

One-Click Clean JSON Saver does not collect, sell, share, or transmit personal data.

## Data Handling

The extension captures JSON API responses only inside the user's browser so the user can preview, copy, save, or export those responses.

Captured responses are stored in extension memory per browser tab and are removed when the tab is closed or when the user clicks **Clear**.

User settings, such as default search text and capture preferences, are stored locally with `chrome.storage.local`.

## Network Usage

The extension does not send captured API responses, settings, browsing history, or user data to any backend server.

## Permissions

The extension requests access to webpages so it can detect `fetch` and `XMLHttpRequest` JSON API responses on pages where the user wants to use the tool.

The extension requests the downloads permission so users can save cleaned JSON files and ZIP exports.

## Contact

For questions, open an issue in the project repository.
